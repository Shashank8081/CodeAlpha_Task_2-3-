# 📊 Population Data Analysis & Visualization Project

## 📌 Project Overview
This project demonstrates the complete data analysis pipeline using real-world population data. It includes data cleaning, exploratory data analysis (EDA), visualization using Python, and dashboard creation using Power BI.

---

## 🎯 Objective
- Clean and preprocess raw data collected from web scraping  
- Perform Exploratory Data Analysis (EDA)  
- Generate meaningful insights  
- Visualize data using charts and dashboards  

---

## 🛠 Tools & Technologies
- Python 🐍  
- Pandas  
- Matplotlib  
- Power BI 📊  

---

## 📥 Data Source
- Dataset extracted from Wikipedia (Task 1: Web Scraping)

---

## ⚙️ Workflow

### 🔹 Task 2: Data Cleaning & EDA (Python)

#### 1️⃣ Data Cleaning
- Removed commas and unwanted symbols from population data  
- Handled missing values using `dropna()`  
- Converted population column to numeric format  

#### 2️⃣ Data Transformation
- Selected relevant columns:
  - Country  
  - Population  
  - Percent  
  - Date  

#### 3️⃣ Exploratory Data Analysis (EDA)
- Checked dataset structure using:
  - `info()`  
  - `describe()`  
- Identified:
  - Top 10 most populated countries  
  - Lowest populated countries  

#### 4️⃣ Data Visualization
- Bar Chart → Top 10 populated countries  
- Histogram → Population distribution  

---

### 🔹 Task 3: Dashboard & Visualization (Power BI)

#### 📊 Dashboard Features
- Interactive charts for population analysis  
- Top & bottom countries comparison  
- Population distribution insights  
- Clean and user-friendly layout  

#### 📈 Key Insights
- Population distribution is highly uneven  
- A few countries contribute a major share of global population  
- Visual dashboards simplify complex data

- Python visualizations (bar chart & histogram)  
- Interactive Power BI dashboard
